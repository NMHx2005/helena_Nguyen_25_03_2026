# Enzo Editing Landing Page

High-end portfolio and landing page for real estate photo and video editing services.

## Tech Stack

- Next.js (App Router) + React 19 + TypeScript (strict)
- Tailwind CSS v4 + shadcn/ui
- Motion (Framer Motion), Lenis smooth scroll, GSAP
- Zod + React Hook Form
- Lucide icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

- `app/` - routes, layout, metadata
- `components/ui/` - shadcn components
- `components/sections/` - landing page sections
- `components/providers/` - app-wide providers
- `components/shared/` - reusable custom primitives
- `lib/` - utilities and helpers
- `public/images/` - static image assets
